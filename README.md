# Data Visualization For Diabetes

Visualize diabetes prevalence data for all countries from 2011, 2013, and 2015.  
Built using Google's [WebGL globe](https://github.com/dataarts/webgl-globe) and diabetes data from [Diabetes Atlas](http://www.diabetesatlas.org/).
